# 🛍️ Sephcocco Lounge – User Interface

This repository contains the **user-facing interface** for **Sephcocco Lounge**, a modern e-commerce web application designed to deliver an elegant shopping experience.

---

## 📌 Project Description

**Sephcocco Lounge** is a stylish and intuitive online platform where users can explore, search, and purchase a curated collection of fashion and lifestyle products. This frontend handles all user interactions – from product browsing to authentication, checkout, and more – providing a seamless shopping experience.

---

## ⚙️ Tech Stack

### 💻 Frontend
- **Next.js** – Server-side rendering and routing
- **TypeScript** – Type safety across the project
- **Tailwind CSS** – Utility-first styling
- **ShadCN UI** – Reusable and accessible UI components
- **Lucide React** – Beautiful icon set
- **Context API** – For managing user authentication state

### 📦 Tooling & Configuration
- **ESLint & Prettier** – Code formatting and linting
- **NVM** – Node version management
- **Vercel** – Deployment platform (optional)

---

```bash
# Clone the repo
